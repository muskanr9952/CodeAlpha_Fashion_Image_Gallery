# 🌟 Bold Muse – Responsive Fashion Image Gallery

A modern and responsive **Fashion Image Gallery** developed using **HTML, CSS, and JavaScript** as part of the **CodeAlpha Frontend Development Internship**.

**Bold Muse** showcases elegant fashion collections through a clean, luxury-inspired interface with smooth animations, responsive layouts, and an interactive lightbox experience.

---

## 📌 Project Overview

This project demonstrates the use of HTML, CSS, and JavaScript to build a visually appealing image gallery. It is fully responsive and provides an interactive browsing experience with multiple fashion categories and image navigation.

---

## ✨ Features

* Responsive design for desktop, tablet, and mobile devices
* Luxury-inspired fashion gallery interface
* Sticky navigation bar
* Hero landing section
* Multiple fashion categories:

  * Women Collection
  * Street Fashion
  * New Arrivals
  * Premium Collection
* Interactive Lightbox View
* Previous & Next Image Navigation
* Smooth hover effects and transitions
* Modern and clean UI/UX
